# vuetifybase

This is a cleaned base to clone if you want to use Vue2 with vuetify  
This base includes :   
- Vue-router
- Vuex

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
